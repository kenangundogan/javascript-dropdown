# Javascript Dropdown
Javascript dropdown - tiny and simple.

### View
```html
<div class="dropdown-container">
    <!-- Button trigger modal -->
    <button 
        class="dropdown-btn"
        data-target="dropdown-default"
    >
    Default Dropdown
    </button>

    <!-- Dropdown List -->
    <div 
        class="dropdown-list" 
        id="dropdown-default"
        >
        <li class="dropdown-item"><a href="#">...</a></li>
        ...
    </div>
</div>
```
